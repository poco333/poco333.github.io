Host Introduction Renewal

ホスト紹介ページ リニューアル

担当

UI / UX デザイン

期間

2020年2月〜3月

ドッグホスト紹介ページは、飼い主さんがどんなドッグホストさんなのかを確認したり、あるいはどの方にお願いするのかを検討するためのページです。 人対人のサービスでは、相手を知るためのとても重要なページです。今回、ドッグホストさんをより魅力的に紹介できるページにするべく改修をおこないまいした。

リニューアルしたデザイン（プロトタイプ）
現状のホスト詳細ページ

Introduction

リニューアル背景

現状の問題として、カテゴリ化できる情報が点在していたり、 ホストの人柄を伝える重要な情報が隠れ、魅力が伝わりきれていない状態でした。

大事な愛犬をお世話する、人対人のサービスとしてホストの魅力が伝わり、飼い主さんが安心できる情報設計とは何か を改めて考え直す必要がありまりました。

Approach

ペルソナの把握

まずは、改めてペルソナの把握を行いました。

ターゲットは主に30後半〜50代辺りの男女です。DogHuggyが「街のみんなが犬の家族」という状態を目指しているように、 ターゲットの価値観も「家族」「共感」「ライフスタイル」「思いやり」などを大事にしている人々に絞られています。

カテゴライズ化

次にターゲットの人達が何を基準に利用の検討をするのか調査し、利用へのブロックとなりそうな心理的ハードルを4つのカテゴリに分類しました。

人となり

・一般家庭に愛犬を預けるとなると、本当に信頼できる人なのか心配になる

・愛犬が心を開く相手だろうか

・飼い主同士も相性や考え方が合うのだろうか

・飼育経験が豊富な人や雰囲気が良い方に頼みたい

・etc

実績

・飼育経験や預かり経験はあるのかな

・利用された方の評価で決めるところもある

・何かしらわんちゃんに関する仕事をされている方なら尚安心なのだけれど…

・etc

環境

・部屋は清潔で普段通り過ごせそうな環境なのかな

・家の種類も気になる（マンションか一軒家か）

・先住犬がいると、相性もあるので心配するし避けたい

・お子さんがいるのかどうかも気になる

・etc

利便性

・駅チカや家から行けそうなら検討するが、遠ければ断念する

・送り迎えがあれば利用したいが、最終的にはその人との相性で決めたい

・etc

Concept

方針の決定

DogHuggyは「街のみんなが犬の家族」というビジョンもさることながら、 人対人のパーソナルなやりとり且つ愛犬を預けるというセンシティブなサービスでもあるため、 4つのカテゴリ面からホストさんの魅力を伝えることを優先し、全体の信頼感（安心感）の向上を目指しました。

ホストさんを「①人となり ②実績 ③環境 ④利便性」の面から魅力を伝え、信頼感（安心感）の向上を目指す

判断の優先順位もそのまま①〜④とする（必要に応じて柔軟に入れ替える）

上記方針を念頭に置き、バラバラに点在している情報をカテゴリ分けで整理し、どう配置していくべきか話し合いました。

Output

UI Change

人の良さを伝えるために

メッセージ

ホスト自身を伝えるアピール文（冒頭文）をより目立たせ、暖かみが出るようにメッセージとしてまとめて差別化しました。

詳細

非表示になっているホスト紹介文は「詳細」のカテゴリにまとめていつでも遷移できるようにしました。

また、非表示になっている情報内容を整理し、ホスト紹介文の文脈から外れる or 外部化した方が良い情報は省くことにしました。

残す情報

飼育経験　/　犬に関する資格　/　家族構成　/　住環境　/　周辺環境

省く情報

ドッグホストになった理由　/　先住犬　/　アクセス　/　送迎について　/　注意点　/　最後に

その他

タイトルについて

現状のタイトルの課題点として、以下が挙げられていました。

・編集はこちらで管理しているため、手間がかかる

・一文が長いと、アピールし切れないホストさんもいる

・タイトルというより、アピール項目としての要素が強い

そこで、ホストさん自身で予め用意したアピールポイントの選択肢から複数選んでもらえるようタグ化し、 少なくとも3〜4つは表示できるようにしました。

因みに駅名などもタイトルに含まれていましたが、タグでは表示させず検索結果一覧でテキスト情報として記載することにしました。

実績面

バッジの追加

ホストさん自身の実績が分かるよう、バッジの追加をしました。

付与するバッジはホストになる前のオンラインテストの結果を基に判断することにしました。

預かり実績の追加

元々なかった情報でしたが、犬種や預かり中のレポート写真を実績として追加しました。

どういった犬種を預かってきたかなど、今までプライベート感が強かった情報を 見える化することで実際の様子や経験を伝える施策です。

その他

口コミについて

以前は、口コミが同じ画面内に一覧で並んでいましたが、今回の改修で情報量が多くなったため、 別画面内で確認できるよう変更しました。

また、口コミがまだないホストさんへは「運営事務局からのコメント」として載せることで 実績のないホストさんであっても、飼い主さんが依頼する際の参考になるよう配慮しました。

環境を伝えるために

住まい

以前は非表示で隠れていた「住まい」に関する情報を見える化する方向で考えていたのですが、 情報が多くなり、必須な情報（プラン、予約日など）にアクセスしにくくなる点、 詳細カテゴリ内を読む際に、続きを読むをタップした後、飼育経験や家族構成などに目を通す流れで住環境や周辺環境にも 目を通せた方がスムーズだと判断し、以前と同じ箇所に配置することにしました。

先住犬

先住犬については、以前だとバラバラな箇所に情報が配置されていたので、統一して載せることにしました。

また、先住犬の存在は検討する際の大きな要因でもあるので、メッセージやホスト紹介文の次に配置しています。

必須な情報を伝えるために

適するタイミングでの提供

必須な情報については、アクセスや予約日もホストさんと連絡をとった上で分かる、相談できる仕様でもあるため、ホスト紹介文（詳細）を優先して下に配置しています。 ただ、比較や検討する際にすぐにアクセスできた方が良い点も考え、ページ内遷移のナビゲーションでカバーすることにしました。

また、例えば事前面談リクエスト時やチャット画面に予約可能日を表示したりなど適するタイミングで情報を提供する方向で改善していくことにしました。
